Project Title

Bowling Star

📌 Project Overview

Bowling Star is an interactive mini bowling alley that combines physical gameplay with electronics and coding. The system uses an ESP32, sensors, an OLED display, LED lighting, and a servo motor to simulate a bowling experience in a compact tabletop format. When pins are knocked down, the system detects the score, displays it live, provides lighting feedback, and resets the setup for the next round.
